Про наш проект

Kroko Sound - це сучасний веб-музичний плеєр, створений для комфортного прослуховування музики. Проект розроблено в рамках курсу WEB PRO.

3000+

Робочих функцій

🎵 Основні можливості

👤
Система користувачів
Реєстрація, вхід, профіль, видалення акаунту
🎵
Музичний плеєр
Play/Pause, Next, Prev, прогрес-бар
🔊
Гучність та швидкість
Регулювання звуку та темпу відтворення (0.5x-2x)
📁
Плейлисти
Створення, редагування, видалення, експорт/імпорт JSON
❤️
Обране
Збереження улюблених треків з візуальним сердечком
📊
Статистика
Лічильник прослуховувань для кожного треку
🔀
Shuffle
Перемішування треків у випадковому порядку
🔁
Repeat
Повторення одного треку по колу
🎨
Зміна теми
6 кольорових схем (фіолетова, синя, зелена, оранжева, рожева, червона)
🔍
Пошук
Фільтрація треків за назвою або артистом
💾
Експорт/Імпорт
Збереження та завантаження плейлистів у JSON форматі
📂
Drag & Drop
Перетягування MP3 файлів для додавання в бібліотеку
⌨️
Гарячі клавіші
Пробіл (пауза), стрілки (перемикання), ↑↓ (гучність)
📱
Адаптивний дизайн
Коректне відображення на ПК, планшетах та телефонах
🎭
Хвильова анімація
Візуалізація музики під час відтворення

👨‍💻 Про проект

Проект виконаний у рамках курсу WEB PRO. Всі функції реалізовані самостійно з використанням сучасних веб-технологій. Kroko Sound — це повноцінний веб-музичний плеєр з можливістю створення плейлистів, збереження улюблених треків, статистикою прослуховувань та персоналізацією інтерфейсу.
