Про наш проект

Kroko Sound - це сучасний веб-музичний плеєр, створений для комфортного прослуховування музики.

Можливості:

  • ✅ Система реєстрації та входу
  • ✅ Персональне збереження даних
  • ✅ Вибір кольору аватарки
  • 🎵 Музичний плеєр з керуванням
  • 📁 Створення плейлистів
  • ❤️ Збереження улюблених треків
  • 📊 Статистика прослуховувань
  • 🎨 Зміна теми оформлення
  • 🔀 Перемішування треків
  • 💾 Експорт/Імпорт плейлистів
  • 📂 Drag & Drop MP3
  • 🔍 Пошук по треках

Технології:

  • 📄 HTML5
  • 🎨 CSS3 (Flexbox, Grid, Адаптив)
  • 💻 JavaScript (ES6+)
  • 💾 LocalStorage
